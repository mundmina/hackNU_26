# KZ8A & KZ4AT Technical Specifications

Sources: Aslanov, Yershov, Zinulla (2015), Novosti Nauki Kazakhstana No. 4 (126); Wikipedia KZ8A article (confirmed).

## KZ8A (Freight Electric Locomotive)

| Parameter | Value |
|---|---|
| Axle arrangement | 2x(2o-2o) — 8 axles, twin-section |
| Power output | 8,800 kW |
| Starting tractive effort | 833 kN |
| Maximum speed | 120 km/h |
| Supply voltage | 25 kV AC, 50 Hz |
| Track gauge | 1,520 mm |
| Traction motors | AC asynchronous (induction) |
| IGBT converters | 3.3 kV / 1,200 A |
| Battery | Nickel-cadmium |
| Operating temp range | -50 to +50 C (Wikipedia) |
| Regenerative braking power | 7,600 kW |
| Maximum braking force | 500 kN |
| Power factor | 0.98 |
| Can replace | 3 sections of VL-80 for trains up to 9,000 t |
| Multi-unit | Up to 3x(2o-2o) under one driver |
| Overhaul interval (KR2) | 3,000,000 km (3x longer than competitors) |
| Energy savings | ~20% vs competitors |

## KZ4AT (Passenger Electric Locomotive)

| Parameter | Value |
|---|---|

| Power output | 4,800 kW |

| Maximum speed | 200 km/h |
| Axle load | 21.5 t |

## TE33A (Diesel-Electric Locomotive)

Source: GE Transportation / Wabtec product data, KTZ-LKZ press releases.

| Parameter | Value |
|---|---|
| Prime mover | GE GEVO-12 (12-cylinder) |
| Power output | ~3,356 kW (~4,500 hp) |
| Axle arrangement | Co'Co' — 6 axles |
| Maximum speed | 120 km/h |
| Starting tractive effort | ~512 kN |
| Weight | ~192 t |
| Track gauge | 1,520 mm |
| Operating temp range | -40 to +45 C |
| Control system | GE Brightstar microprocessor |
| Traction motors | 6x GE 5GEB30 AC induction |
| Emissions | US EPA Tier 2 equivalent |
| Telemetry | GE Trip Optimizer / GoLINC |

## Key Advantages (from paper)

- KZ8A maintenance intervals are 3x longer than VL80t / 2ES5K
- 10% higher design speed than comparable models
- 20% energy savings due to asynchronous drive + regenerative braking
- Can haul 30% heavier trains than VL80 series
